#### What is this project?
GameBoa is a Game Boy emulator written in Python.

#### Project goals
- [ ] GUI
- [ ] Debugger
- [X] ROM loading
- [ ] ROM patching
- [ ] CPU emulation
- [ ] GPU emulation
- [ ] Input emulation
- [ ] Sound emulation
- [X] Memory emulation
- [ ] Link cable support
- [ ] Save state support
- [ ] Neural network support
