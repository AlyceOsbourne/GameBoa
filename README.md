#### What is this project?
GameBoa is a Game Boy emulator written in Python.

- [ ] CPU emulation (WIP)

- [ ] Input emulation (WIP)
- [ ] Sound emulation
- [ ] Memory emulation (WIP)
- [ ] Link cable support
- [ ] Save state support
- [ ] Neural network support
